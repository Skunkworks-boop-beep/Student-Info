## Third-party software

This application builds on open-source libraries and tools, including **React**, **Vite**, **Tailwind CSS**, **React Router**, **Leaflet** / **react-leaflet**, **Recharts**, **Radix UI** primitives, **Lucide** icons, **date-fns**, **Motion**, and others listed in `package.json`. See each package’s license in `node_modules` or on npm.

## Branding note

The codebase uses **generic** institution copy (e.g. demo emails, campus-neutral wording) so deployments can be white-labeled. **Team and credit attribution** are documented here and are not tied to a single university’s branding in the UI.
